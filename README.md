Stratis Enterprise Analytics Dashboard
A responsive, cleanly designed front-end dashboard built to monitor multi-channel sales environments. This UI provides a unified view of e-commerce operations, specifically optimized for tracking both digital storefronts (like Amazon) and physical retail outlets.
🚀 Features
Clean, Light-Themed UI:** A minimalist white and gray aesthetic that prioritizes data readability and a professional user experience.
Smooth Animations:** Implements CSS-based fade-ins and slide-up animations for a dynamic, modern feel upon loading.
Cross-Channel Tracking:** Dedicated metric cards for total revenue, online orders, and retail counter conversions.
Custom UI Components:** * Multi-color task progress charts (e.g., Inventory Synchronization, Package Dispatch).
Dynamic status badges for order monitoring (Completed, Pending, Cancelled).
Responsive data tables mapping out client channels, including specific QR sign payments.
Interactive Navigation:** Toggleable sidebar menu and a persistent search/notification header.
🛠️ Built With

* HTML5
* CSS3 (Custom styling, CSS Grid/Flexbox)
* Vanilla JavaScript (Counter animations, UI toggles)
* FontAwesome (Icons)
* Google Fonts (Inter)

## 💻 Getting Started

To view the dashboard locally:
1. Clone this repository: https://github.com/sumanta387/stratis-ecommerce-dashboard.git
2. Open the project directory.
3. Open `index.html` in your preferred web browser. No local server is required.
